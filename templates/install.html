Step 1

Trust Local HTTPS

Required
1 Download the local CA

Use the button above on your phone. This is the local certificate authority that signed MIRA's HTTPS certificate.

2 Install and trust it

On iPhone: install the profile, then go to Settings > General > About > Certificate Trust Settings and enable full trust.

3 Reopen MIRA

After trust is enabled, reopen {{ host_url }}. The privacy warning should be gone before you add it to your Home Screen.

Step 2

Install From the Right Browser

Phone
iPhone SafariBest on iOS

Open MIRA in Safari, tap Share, choose Add to Home Screen, then enable Open as Web App if iOS shows it.

iPhone ChromeUse Safari

Chrome on iPhone can add a shortcut, but Safari is the reliable path for a true iPhone web app icon and standalone launch.

Android ChromeNative Prompt

Tap Install App in MIRA. If Chrome does not show the prompt yet, open the Chrome menu and choose Install app or Add to Home screen.

Other BrowsersFallback

Use your browser's share or menu actions to add MIRA to the Home Screen. If iPhone behavior is limited, switch to Safari.

Step 3

Refresh Old Installs

Clean Retry
A Delete older shortcuts

Remove any previous MIRA icon that only showed a letter or opened inside a normal browser tab.

B Reload after trust

Open MIRA again after the certificate trust step so the browser sees the latest manifest, icons, and service worker cache.

C Add it again

Re-add MIRA from the browser you want to use. The installed app should launch in standalone mode and use the branded icon.
